# Add-Christmas-Hat

给人像图片添加圣诞帽
功能介绍
圣诞将至，为图片中的每一个人添加一顶可爱的圣诞帽。
使用指南
准备好待处理的图片文件（支持 JPG、PNG 等格式）。
运行代码。
等待数秒，添加好圣诞帽的图片会自动保存。
注意事项
确保图片中的人脸清晰可见，模糊不清或遮挡严重的人脸会影响圣诞帽的佩戴精准度。

Adds a Christmas hat to the people in the picture
Feature Overview
With Christmas just around the corner, this tool adds a cute Christmas hat to every person in your picture.
Usage Guide
Prepare the image file to be processed (supports formats such as JPG and PNG).
Run the code.
Wait a few seconds, and the image with Christmas hats added will be saved automatically.
Notes
Make sure that the faces in the image are clearly visible. Blurred or heavily occluded faces will affect the accuracy of hat placement.
